## 🪐 项目初衷

Vibe Coding降低了非技术人员进行创作的门槛。我希望用这份可视化的网页，向您更加全面地介绍自己。
如果能让您从中感知到我更多的切面，如果能让您感到一丝的触动、一丝的波澜，这就是 AI 带给我的最大价值。


---

## ✨ 叙事主题设计思路

整个网页以「小行星」为核心叙事意象，构建一段完整的宇宙旅程：

| 页面 | 主题 | 叙事含义 |
|------|------|----------|
| 第一页 | 行星诞生 | 我是谁——初次相遇 |
| 第二页 | 星光积攒 | 我从哪里来——教育积累 |
| 第三页 | 星轨初具 | 我走过哪些路——项目经历 |
| 第四页 | 星辰涌现 | 我创造了什么——实践成果 |
| 第五页 | 星轨归处 | 再次相遇——感谢相逢 |

视觉上以粉紫蓝星空为底，星球、光球、星轨弧线贯穿始终，让您在浏览的过程中感受到一段完整的「宇宙旅程」，而不只是在翻阅一份简历。

## 📖 各页面设计说明

### 第一页 · 行星诞生
打字机渐显动效逐行呈现自我介绍文案，文字以双层text-shadow（微弱深色投影+白色光晕）在复杂星空背景上保持清晰可读。两个交互按钮——「星图下载」触发简历PDF下载，「星轨相接」弹出联系方式弹窗。

### 第二页 · 星光积攒
以一条自下而上逐渐发光的星轨线为视觉核心，标注安徽与上海两个节点，点击节点弹出Glassmorphism毛玻璃卡片，展示对应的教育经历。

### 第三页 · 星轨初具
以个人为中心向外扩散的同心圆轨道，承载百度、美团、Lovart三段工作经历。hover轨道高亮，点击弹出半屏侧拉卡片展示详细项目内容。

### 第四页 · 星辰涌现
三个带行星环的发光球体，初始聚拢，hover后以弹性动画散开。点击进入左侧条目列表+右侧阅读区的双栏阅读界面，展示Vibe Coding记录、产品分析文档、AI学习理解三类成果内容。

### 第五页 · 星轨归处
复用第一页的视觉结构与动效，替换为收尾文案，与开篇形成叙事闭环。「星图下载」与「星轨相接」再次出现，赋予更深的叙事含义。

## 🤖 工具使用及思考

这个项目从设计到实现，全程借助AI工具完成，以下是我的使用方式：

**1. 用Claude做架构决策**
在正式 Vibe Coding 工作之前，我先与 Claude 进行反复的沟通与交流，最终落地可实现的开发文档。

**2. 用Trae做代码实现**
开发文档确认后，我在Trae IDE中与AI协作落地。在与Trae的协作中，我的叙事概念始终保持统一。

**3. 迭代而非一次成型**
没有任何一个页面是一次提示词就完成的。我的每一页实现方式、交互形式以及内容的组织，都开始产生了新的变化。我在一次又一次的迭代中，探索最佳的实现方式。

**4. 核心认知**
当 Web Coding 降低了人们创作的门槛，当这份工具被摆在越来越多的人面前，我们不再能够以工具的难度来人为地给自己设置障碍，或者阻碍自己进入某一领域。
创作的难度逐渐降低，但决定创作质量的依旧是：
1. 对行业技术的认知
2. 对审美艺术的感知
3. 对世界与自我的思考

人的逻辑、判断和审美能力，在 AI 时代被不断地放大。

在这样的一个时代，一个人的灵魂是最重要的。

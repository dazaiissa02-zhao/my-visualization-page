# 第一页视觉与动效重构计划

感谢您提供视频素材！这正是我们需要的关键元素，它将彻底解决“动静分离”的问题，带来更高级的视觉体验。

## 核心变更
1.  **背景升级**: 废弃原本的静态图片 `planet-bg.jpg`，启用您新上传的 `第一页.mp4` 作为动态全屏背景。
2.  **粒子移除**: 根据您的建议，彻底移除 `ParticleSphere`（那个试图包裹星球的 3D 粒子层）。既然视频自带了星球的动态，我们就不再画蛇添足，保持画面的纯净。
3.  **排版重构**:
    *   **布局**: 将文字内容从中心移动到左侧 (`items-start`, `text-left`)，使其悬浮在云层上方，避开右侧的星球视觉重心。
    *   **字体**: 引入更优雅的衬线字体（Serif）用于标题，提升文学感和精致度。

## 实施步骤

### 1. 资源处理
*   将 `assets/第一页.mp4` 移动到 `src/assets/planet-bg.mp4`。

### 2. 代码重构 (`src/pages/Page1.jsx`)
*   **背景层**: 将 `div` 背景图替换为 `<video>` 标签。
    *   设置 `autoPlay`, `loop`, `muted`, `playsInline` 确保自动播放。
    *   使用 `object-cover` 确保视频全屏填充。
*   **3D 层**: 删除 `<Canvas>` 及其相关的 `ParticleSphere` 组件代码。
*   **内容层**:
    *   容器布局：从 `items-center text-center` 改为 `items-start text-left`。
    *   位置调整：增加左侧内边距 (`pl-12` 或 `pl-16`)，调整上边距以适应云层位置。
    *   字体样式：
        *   标题：使用 `font-serif` (需在 Tailwind 配置中确认或直接引入 CSS 字体)，增加字重和字间距。
        *   正文：保持轻盈，增加行高。

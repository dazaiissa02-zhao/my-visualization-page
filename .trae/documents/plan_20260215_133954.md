# 英文字体柔和化调整与自定义指南

您觉得 `Cinzel` 过于锐利（Sharp），这很正常，因为它是一款衬线装饰体。为了解决这个问题，我们将切换到更柔和、更现代的 **Montserrat (Thin/Light)**，并为您提供详细的修改指南。

## 解决方案

### 1. 字体调整 (Font Switch)
*   **目标**: 降低锐度，增加柔和感。
*   **新选择**: **Montserrat**。
    *   这是一款几何无衬线体，圆形结构非常多，视觉上非常圆润、亲和。
    *   我们将使用它的 **Thin (100)** 或 **ExtraLight (200)** 字重，以达到您想要的“纤细”效果。

### 2. 自定义指南 (How to Customize)
您想知道在哪里修改字体和大小。核心文件依然是 `src/pages/Page1.jsx`。

#### (a) 修改字体 (Font Family)
在 `Page1.jsx` 中，找到 `lines.map` 渲染部分：

```javascript
style={
  index === 0 || index === 1
    ? { fontFamily: '"Montserrat", sans-serif', fontWeight: 200 } // <--- 改这里 (主标题英文)
    : { fontFamily: '"Noto Serif SC", serif' } // <--- 改这里 (副标题中文)
}
```

*   **配置位置**: `index.html` 的 `<head>` 中引入了字体链接。
    *   如果您想换其他字体，先去 Google Fonts 选好，把链接贴到 `index.html`，然后把名字填入上面的 `fontFamily`。

#### (b) 修改大小 (Font Size)
在 `className` 字符串中修改 Tailwind 类名：

```javascript
className={`drop-shadow-md ${
  index === 0
    ? 'text-5xl md:text-7xl ...' // <--- 改这里 (主标题第一行) text-5xl 是移动端，md:text-7xl 是桌面端
    : index === 1 
      ? 'text-5xl md:text-7xl ...' // <--- 改这里 (主标题第二行)
      : 'text-lg md:text-2xl ...'  // <--- 改这里 (副标题)
}`}
```

*   **数值参考**:
    *   `text-4xl`: 36px
    *   `text-5xl`: 48px
    *   `text-6xl`: 60px
    *   `text-7xl`: 72px
    *   `text-8xl`: 96px

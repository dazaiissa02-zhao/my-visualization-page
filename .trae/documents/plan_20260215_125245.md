# 第一页细节优化与功能扩展计划

我们收到您的四个关键反馈，将逐一进行精准修复与功能实现。

## 问题分析与解决

### 1. 文本分段与去逗号 (Text Formatting)
*   **需求**: “你好，我是赵晓婷” -> 分段显示 + 去除逗号。
*   **实现**: 将这一行拆分为两行：
    *   第一行: “你好”
    *   第二行: “我是赵晓婷”
    *   或者直接去掉逗号并在同一行内用空格或换行符分隔。根据“分段”描述，我理解为视觉上的两行或明显的断句。为了视觉平衡，建议改为两行大标题。

### 2. THREE.BufferGeometry NaN 错误 (Fix NaN Bug)
*   **原因**: `maath/random` 的 `inSphere` 函数生成的 `Float32Array` 可能包含 `NaN` 值，或者传入的数组长度/步长与 buffer attribute 不匹配。这通常是因为 `maath` 生成数据时某些数学计算越界或初始数组未正确填充。
*   **解决**: 
    *   检查 `random.inSphere` 的调用参数。确保 `radius` 是有效数字。
    *   或者添加数据校验，过滤掉 NaN 值。
    *   更稳妥的方式：改用标准的三维坐标生成逻辑（手动循环生成），确保数据绝对安全，不再依赖可能不稳定的第三方库函数，或者确保 `maath` 使用正确。

### 3. 按钮突兀上移 (Fix Button Jump)
*   **原因**: 按钮上有 `hover:scale-105` (Tailwind) 或 Framer Motion 的 `whileHover={{ scale: 1.05 }}`。当鼠标悬停时，整体放大导致视觉上的“跳动”。
*   **解决**: 移除 `scale` 缩放效果，改为更柔和的背景色变化 (`hover:bg-white/20`) 或边框发光，保持布局位置的绝对稳固。

### 4. 新增“星轨相接”联系按钮 (New Contact Feature)
*   **UI 设计**: 
    *   在“下载简历”旁新增一个风格一致的玻璃态按钮。
    *   图标: 使用电话图标 (`Phone` from `lucide-react`)。
    *   文案: “星轨相接”。
*   **交互逻辑**:
    *   点击后弹出一个半透明模态框 (Modal)。
    *   模态框内容: 显示电话号码 `19855555555`，并提供“复制”或“拨打”功能。
    *   动画: 模态框需有淡入淡出效果，背景模糊 (`backdrop-blur`)。

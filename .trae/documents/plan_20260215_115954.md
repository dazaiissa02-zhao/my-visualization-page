# HeroBackground 组件开发计划

您希望创建一个融合了“视频背景”与“3D 粒子”的 `HeroBackground` 组件，这其实是结合了我们前两个阶段的优点：既有视频的真实感，又有 3D 代码的交互性。

## 技术方案

### 1. 依赖安装
*   您提供的代码中使用了 `maath` 库来生成随机粒子坐标，这是一个轻量级的数学库，我们需要先安装它：`npm install maath`。

### 2. 组件创建 (`src/components/HeroBackground.jsx`)
*   我们将完全参考您提供的代码结构，并做以下微调以适配当前项目：
    *   **粒子颜色**: 使用淡粉色 (`#ffd1dc`) 和浅蓝色 (`#add8e6`) 的混合，或者通过 `vertexColors` 实现渐变，以呼应插画色调。
    *   **粒子动态**: 添加 `useFrame` 让粒子层缓慢旋转，增强呼吸感。
    *   **位置参数**: 将 `group` 的 `position` 暴露为 props 或根据当前视频的星球位置（右上角）进行硬编码调整 (`[3.5, 2.2, 0]`)。

### 3. 页面集成 (`src/pages/Page1.jsx`)
*   将原本直接写在 `Page1.jsx` 里的视频和内容逻辑剥离。
*   引入 `HeroBackground` 组件作为底层容器。
*   **内容插槽**: 将原本的“你好，我是赵晓婷”等文字内容作为 `children` 传递给 `HeroBackground`，或者直接在 `HeroBackground` 内部保留一个 `children` 插槽，保持组件的复用性。

## 实施步骤
1.  **安装依赖**: `npm install maath`。
2.  **创建组件**: 新建 `src/components/HeroBackground.jsx`。
3.  **重构页面**: 修改 `src/pages/Page1.jsx` 使用新组件。
4.  **微调效果**: 调整 Canvas 的 `camera` 位置和 `StarParticles` 的坐标，确保粒子精确覆盖在视频中的星球上。

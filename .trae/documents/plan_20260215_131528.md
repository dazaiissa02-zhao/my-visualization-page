# 副标题字体艺术化升级计划

您希望副标题更具“艺术感”。目前的字体可能过于规整（sans-serif），缺乏个性和油画的韵味。

## 解决方案

### 1. 字体选择 (Font Selection)
*   **当前**: 默认 sans-serif (无衬线)。
*   **目标**: 引入更有气质的 **衬线体 (Serif)** 或 **手写体 (Handwriting/Calligraphy)**。
    *   **方案 A (推荐)**: 使用 Google Fonts 引入一款优雅的中文字体，如 **"Noto Serif SC" (思源宋体)** 或 **"Zhi Mang Xing" (织其芒星 - 书法体)**。考虑到副标题较长且需要易读性，**思源宋体 (Noto Serif SC)** 是最安全且优雅的选择，既有书卷气又不失清晰。如果想要更飘逸，可以尝试 **"Ma Shan Zheng" (马善政毛笔楷书)**。
    *   **方案 B (保守)**: 直接使用 CSS `font-serif`，但这依赖于用户系统字体，效果不可控。

### 2. 实施步骤
1.  **引入字体**: 在 `index.css` 或 `index.html` 中引入 Google Fonts 链接。
    *   选择: `Noto Serif SC` (权重 300, 400, 700)。
2.  **应用样式**: 在 `Page1.jsx` 中，将副标题的 `className` 从 `font-light` 修改为 `font-serif` (如果已配置 Tailwind theme) 或直接内联 `style={{ fontFamily: '"Noto Serif SC", serif' }}`。

## 建议
鉴于“油画”主题，我建议使用 **"Noto Serif SC"**。它比黑体更具人文气息，又比草书更易读，非常适合作为副标题及其说明文字。

请确认：您偏向于**优雅的宋体**（易读、文艺）还是**狂野的书法体**（艺术感强、但小字难认）？默认我将为您配置 **Noto Serif SC (宋体)**。

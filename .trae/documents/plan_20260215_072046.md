# 个人可视化网页前三页实现计划

根据您的需求和提供的 PRD 文件，我为您规划了前三页的开发方案。每一项技术选型和设计决策，我都会在后续的 `FORTrae.md` 文档中详细解释“为什么这么做”，确保您不仅获得代码，还能获得思路。

## 核心技术栈 (Technical Stack)
- **框架**: React + Vite (轻量、快速，适合个人项目)
- **样式**: Tailwind CSS (原子化 CSS，便于快速实现 Glassmorphism 毛玻璃效果)
- **3D 视觉**: Three.js + React Three Fiber (R3F) (实现“行星”粒子球体和星轨的核心库)
- **动画**: Framer Motion (处理文字打字机效果、页面切换和 UI 组件的平滑过渡)
- **滚动**: CSS Scroll Snap 或 `react-use-scroll-snap` (实现全屏吸附滚动)

## 详细实施步骤 (Implementation Steps)

### 第一阶段：项目初始化与基础设施
1.  **搭建脚手架**: 使用 Vite 创建 React 项目，配置 Tailwind CSS。
2.  **安装依赖**: 引入 R3F (`@react-three/fiber`, `@react-three/drei`), Framer Motion, 路由等。
3.  **全局布局**: 设置全屏滚动容器 (Scroll Snap Container)，确保每一页都能自动吸附。
4.  **文档机制**: 创建 `FORTrae.md`，用于记录每次开发的思考与教学内容。

### 第二阶段：第一页 - 一颗小行星 (The Asteroid)
**设计目标**: 建立第一印象，展示 3D 粒子球体与自我介绍。
1.  **3D 场景搭建**:
    - 创建一个全屏 Canvas。
    - 实现 **粒子球体 (Particle Sphere)**：使用 R3F 的 `Points` 或自定义 Shader，使其缓慢自转。
    - **交互**: 鼠标移动时，粒子产生微弱的斥力或位移，模拟“星空漫游”感。
2.  **UI 层**:
    - **打字机效果**: 使用 Framer Motion 逐字显示“你好，我是赵晓婷...”。
    - **下载按钮**: 设计一个极简的 Icon/按钮，点击触发 PDF 下载。

### 第三阶段：第二页 - 星光积攒 (Education)
**设计目标**: 教育背景，极简地图与星轨动画。
1.  **地图可视化**:
    - 使用 SVG 或 Canvas 绘制极简的华东地区轮廓 (安徽 & 上海)。
    - **星轨动画**: 实现一个光点从安徽坐标沿贝塞尔曲线飞向上海，留下拖尾效果。
2.  **卡片交互**:
    - **交互点**: 在地图上标记“安徽”和“上海”两个节点，带有呼吸灯效果。
    - **毛玻璃卡片**: 点击节点，弹出 Glassmorphism 风格卡片 (背景模糊、半透明白)，展示学历信息。

### 第四阶段：第三页 - 星轨初具 (Experience)
**设计目标**: 工作经历，同心圆轨道与覆盖层。
1.  **轨道系统**:
    - 以屏幕中心（或延续第一页的行星）为圆心，绘制三条同心圆轨道。
    - 在轨道上布置“百度”、“美团”、“Lovart”的节点 Icon。
2.  **交互逻辑**:
    - **悬停 (Hover)**: 鼠标悬停在节点上时，对应轨道高亮发光。
    - **点击 (Click)**: 点击节点，展开全屏或半屏遮罩层 (Overlay)，详细展示工作内容。
    - **关闭**: 点击遮罩层外区域，平滑收起。
